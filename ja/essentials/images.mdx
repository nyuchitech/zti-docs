## 画像

### Markdownの使用

この[Markdown構文](https://www.markdownguide.org/basic-syntax/#images)を使用すると、以下のコードで画像を追加できます

```md
![title](/path/image.jpg)
```

画像ファイルのサイズは5MB未満である必要があることに注意してください。それ以上の場合は、[Cloudinary](https://cloudinary.com/)や[S3](https://aws.amazon.com/s3/)などのサービスでホストすることをお勧めします。そのURLを使用して埋め込むことができます。

### 埋め込みの使用

画像をより柔軟にカスタマイズするには、[埋め込み](/writing-content/embed)を使用して画像を追加することもできます

```html
<img height="200" src="/path/image.jpg" />
```

### iFrame

ドキュメント内に別のHTMLページを読み込みます。主に動画の埋め込みに使用されます。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
