# ジンバブエの料理

![Traditional Zimbabwean sadza with stew](/images/hero-light.svg)

ジンバブエの料理は、国の文化遺産、農業慣行、歴史を美味しく反映しています。主食の穀物、新鮮な野菜、適度な量の肉を中心とした伝統的なジンバブエの食事は、栄養価が高く、風味豊かで、日々の生活のリズムと深く結びついています。

## 伝統的な主食と料理

### 主食

<CardGroup cols={2}>
  <Card title="サッザ" icon="bowl-food">
    ジンバブエ料理の基礎であるサッザは、現地で「ミーリーミール」として知られるトウモロコシの粉から作られた濃厚なポリッジです。ポレンタに似ていますが、より濃厚な食感で、ほとんどのジンバブエの家庭で毎日食べられています。

    **食べ方**：右手で小さな団子状に丸め、付け合わせやシチューにつけて食べます。食器と主食の両方の役割を果たします。

    **種類**：

    * 白いサッザ（精製トウモロコシの粉）- 最も一般的

    * サッザ・レズビヨ（シコクビエ）- 土っぽい風味の伝統的な変種

    * 黄色いサッザ（ソルガムベース）- あまり一般的ではありませんが、栄養価が高い
  </Card>

  <Card title="野菜と付け合わせ" icon="carrot">
    サッザには常に「ウサビ」と呼ばれる付け合わせやシチューが添えられます。以下のようなものがあります：

    * **ムリウォ/デレレ**：コラードグリーンやケールに似た葉物野菜で、トマト、玉ねぎ、ピーナッツバターと一緒に調理されることが多い

    * **ニェブヘ**：クレオメの葉

    * **デレレ**：オクラ、地元の人々が好む粘り気のある食感で調理されることが多い

    * **ムボーラ**：カボチャの葉、玉ねぎとトマトで炒めたもの

    * **ルピザ**：ササゲの葉

    これらの野菜は日々の栄養の要となっており、可能な場合は肉が加えられます。
  </Card>

  <Card title="肉料理" icon="drumstick-bite">
    肉は貴重で、伝統的な環境では特別な機会のために取っておかれることが多いです。

    * **ニャマ・イェ・フク**：鶏肉のシチュー、多くの場合放し飼いで風味豊か

    * **ニャマ・イェ・モンベ**：柔らかくなるまでじっくり煮込んだ牛肉のシチュー

    * **ロードランナー**：地元の放し飼い鶏、商業用の鶏よりも硬いが風味が豊か

    * **マゾンド**：柔らかくなるまでじっくり煮込んだ牛の足（蹄）

    * **オックステールシチュー**：数時間の調理を要する濃厚でゼラチン質のシチュー

    * **カペンタ**：小さな干し魚で、多くの場合戻してトマトと玉ねぎで調理される
  </Card>

  <Card title="ジビエ" icon="bison">
    ジビエはジンバブエの豊かな野生動物の遺産を反映しています。観光地では以下のようなものが見つかるかもしれません：

    * **ニャマ・イェ・ムフカ**：ジビエの一般的な呼称

    * **クーズー**：脂肪分の少ないアンテロープの肉で、マイルドな風味

    * **インパラ**：一般的なジビエで、鹿肉に似た味

    * **ワニ**：多くの場合、魚と鶏肉を思わせる食感の尾のステーキとして提供される

    * **イボイノシシ**：脂肪分が少なく風味豊かで、多くの場合リブやシチューとして調理される

    * **ビルトン**：ジャーキーに似た乾燥・塩漬けの肉で、牛肉やジビエから作られる

    **注意**：ジビエは規制されています。提供する施設が適切なライセンスを持っていることを確認してください。
  </Card>
</CardGroup>

### 特別料理

<Card>
  **探す価値のあるジンバブエの代表的な特別料理：**

  * **ドビ**：肉や野菜を使ったピーナッツバターのシチュー、濃厚でクリーミー

  * **モパネワーム（マドラ）**：モパネの木から収穫されるタンパク質が豊富な幼虫で、乾燥させてから戻して調理する

  * **ムリウォ・ウネ・ドビ**：ピーナッツバターソースで調理した緑葉野菜

  * **マポポ**：青パパイヤを砂糖で煮て作る甘いパパイヤキャンディー

  * **マヘウ**：発酵トウモロコシ飲料、わずかに酸味があり、しばしば食事の代わりとして飲まれる

  * **カボチャのポリッジ**：カボチャ、ミーリーミール、砂糖で作る季節の朝食料理

  * **ンヘジ**：野生のキノコ、特に雨季に豊富に見られる

  * **ズビヨ**：伝統的なシコクビエのポリッジ
</Card>

## 食事の習慣とエチケット

<CardGroup cols={2}>
  <Card title="伝統的な食事の慣習" icon="hands-holding">
    ジンバブエの伝統的な食事では：

    * 食事は**右手のみで食べる**

    * Sadzaは指を使って小さな団子状に丸める

    * 共通の鍋や大皿を共有することが多い

    * 食事前の手洗いは必須で、通常はホストが手伝う

    * 敬意を表して年長者が最初に給仕される

    * 食事中は感謝の気持ちを表現する

    **都市部 vs. 農村部**：これらの慣習は農村部の家庭や文化的なイベントでより一般的です。都市部のレストランでは通常、西洋式のカトラリーを使用します。
  </Card>

  <Card title="もてなしの習慣" icon="people-roof">
    ジンバブエのもてなしは食事の共有を中心としています：

    * 提供された食事を断ることは失礼とみなされる可能性がある

    * 料理人を褒めることは感謝される

    * 家庭訪問時の小さな贈り物は歓迎されるが、必須ではない

    * 客には最高の部分が提供されることがある

    * 「食事はしましたか？」は思いやりを表す一般的な挨拶

    * 予期せぬ訪問者にも可能であれば食事が提供される
  </Card>

  <Card title="レストランでのエチケット" icon="utensils">
    ジンバブエのレストランで食事をする際：

    * サービスがゆっくりな場合がある；忍耐が評価される

    * 観光地ではチップ（10-15%）が慣例

    * 高級店では予約が推奨される

    * ドレスコードは一般的にカジュアルだが、高級レストランではスマートカジュアル

    * シェフへの賛辞は歓迎され、しばしば伝えられる
  </Card>

  <Card title="特別な機会" icon="cake-candles">
    ジンバブエのお祝いでは食事が中心的な役割を果たします：

    * **結婚式の宴**：大量の肉、特に牛肉

    * **葬儀**：コミュニティのメンバーが家族を支援するために食事を持ち寄る

    * **休日の集まり**：日常のsadzaの代わりに特別な料理として米が提供される

    * **収穫祭**：季節の農産物と伝統的なビールが特徴

    * **Braais**（バーベキュー）：グリルした肉を中心とした社交的な集まり
  </Card>
</CardGroup>

## 食事制限と特別なニーズ

<Info>
  **食事制限がある場合の旅行：**

  * **ベジタリアン/ビーガン**：伝統的な野菜料理や付け合わせは良い選択肢ですが、多くは動物性脂肪で調理されています。確実に理解してもらうために「strict vegetarian」と指定してください。

  * **グルテンフリー**：ほとんどの伝統的な食事は自然とグルテンフリーです（サザ、野菜、肉）が、ソースに小麦由来の増粘剤が使用されていないか確認してください。

  * **ハラール/コーシャ**：認証された選択肢は限られています。厳格な要件の場合は自炊が必要かもしれません。

  * **アレルギー**：ピーナッツは料理でよく使用されます（ドヴィ）。アレルギーについて明確に伝え、緊急用の薬を携帯してください。

  **役立つフレーズ：**

  * "Handidye..."（私は...を食べません）

  * "Ndine allergy ye..."（私は...にアレルギーがあります）

  * "Mune... mune izvi?"（これに...は入っていますか？）
</Info>
