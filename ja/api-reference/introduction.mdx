## ようこそ

APIドキュメントを作成する方法は2つあります：[OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup)と[MDXコンポーネント](https://mintlify.com/docs/api-playground/mdx/configuration)。スターターキットでは、以下のOpenAPI仕様を使用しています。

</Card>

## 認証

すべてのAPIエンドポイントは、Bearerトークンを使用して認証され、仕様ファイルから取得されます。

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
