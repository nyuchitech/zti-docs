Stay up to date with the latest improvements to Zimbabwe Travel Information.

<Update label="2026-04-13" description="April 2026" tags={["Feature", "Update", "Fix"]}>
  ## New directory hub

  The [Directory](/directory) is now a dedicated section in the top navigation with its own tab. You can browse verified accommodation, local experts, hospitality businesses, and campsites — all organized into clear categories. New pages include a [Hospitality directory](/directory/hospitality) covering lodges, restaurants, DMCs, wellness centres, and activity providers, plus an expanded [Expert directory](/directory/experts) with category-based browsing.

  ## Verification badges on directory listings

  All directory cards now display verification badges so you can quickly tell which businesses and experts have been vetted. Badge tiers are colour-coded and appear alongside each listing's name, with category labels beneath for easy scanning.

  ## Bus operator details for popular routes

  The [Transportation guide](/essentials/transportation) now includes a route-by-route table of bus operators — covering Harare to Victoria Falls, Bulawayo, Mutare, Kariba, and more. Operators like CAG, ExtraCity Luxury, Intercape, and Pathfinder are listed with estimated journey times and practical tips from local travelers.

  ## Invitation letter guidance for visa applicants

  If you are applying for a Zimbabwe visa from a Category C country, the [Visas and entry page](/essentials/visas-and-entry) now explains how to obtain an invitation letter — from hotels, tour operators, or personal contacts — along with step-by-step tips for using the eVisa portal.

  ## Improved page descriptions for search

  Descriptions across all 116 pages have been audited and rewritten to be unique, concise, and optimised for search engines. This means more accurate previews when you find our pages through Google or other search engines.

  ## Bug fixes

  - Fixed duplicate heading display across all content pages
  - Resolved rendering errors on directory and expert listing pages
  - Fixed broken card layouts on several destination and planning pages
</Update>
